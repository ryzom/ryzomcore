NeL World Editor user guide

Mouse

  • Left button : select - transform in the 3d view
  • Middle button : drag the 3d view
  • Mouse wheel : zoom the 3d view
  • Right button : context menu for the selection

Keyboard short cuts

  • ALT+F4 : exit
  • ALT+F7 : project settings
  • ALT+ENTER : open the properties dialog for the selection
  • C : select children of selection
  • CTRL+A : select all
  • CTRL+C : copy selection
  • CTRL+G : show / hide grid points
  • CTRL+N : new project
  • CTRL+O : open a project
  • CTRL+P : show / hide PACS surfaces
  • CTRL+S : save the current project
  • CTRL+V : paste
  • CTRL+X : cut selection
  • CTRL+Y : redo
  • CTRL+Z : undo
  • D : show / hide details
  • DEL : delete selection
  • E : expand selection
  • F1 : help
  • F2 : tile mode
  • F3 : transition mode
  • F4 : primitive mode
  • F5 : selection mode
  • F6 : translation mode
  • F7 : rotation mode
  • F8 : turn mode
  • F9 : scale mode
  • F10 : add point mode
  • G : show / hide grid
  • H : hide selection
  • L : show / hide landscapes
  • O : show / hide IA collision maps
  • P : show / hide primitives
  • R : collapse selection
  • S : show selection
  • SPACE : toggle in sub object mode
  • Y : show / hide layers

Special directories

  • data path/"collisionmap" : place here the .tga file for the collision map
  • data path/"pacs" : place here your PACS files. (*.rbank, *.lr and *.gr)

follow_route

Follow a route from the start point to the end point.

Parameters:

shape: set a shape for group. could be : 'rectangle'.
ratio: set the ratio of the shape; largeur/longueur as a number. (valid if there is a group shape).
xsize: set the xsize (largeur) space between two bots. (valid if there is a group shape).
ysize: set the ysize (longueur) space between two bots. (valid if there is a group shape).
running tag the group for running.

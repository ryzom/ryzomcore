Teleport_trigger

 

Represents a teleport trigger ( aka lift )

- If the trigger is a child of a continent, it can be accessed from everywhere

- If the trigger is a child of a teleport_destination, it can only be accessed from this destination

 

Properties :

 

name  : name of the trigger. For leveldesign use only

pacs_trigger_id : . This is the trigger id given by the 3D team. It identifies the trigger physically on the landscape

 

destinations  : list of teleport_destinations

 

auto_teleport  : the user uis teleported as he enters the trigger. You’d better put a unique destination in it…

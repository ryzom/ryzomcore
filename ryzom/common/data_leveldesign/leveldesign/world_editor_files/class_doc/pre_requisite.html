pre_requisite

 

This item holder allows to set special conditions that will all the player to select the mission. If he doesn’t meet the requirements the mission isn’t available for the player (icon is disabled in the task giver menu).

 

how to specify more than 1 mission ?

AND (exemple: mission 01 and 02 must be done)
mission_01
mission_02

OR (exemple: mission 01 or mission 02 must be done)
mission_01;mission_02

 

Properties:

 

name: Name of the answer only used by the mission designer.

 

require_brick_knowledge: The player needs to know listed bricks. See item_guidelines for the formatting requirements.

 

require_fame: List of couples Faction/Fame value that indicates the minimum fame value needed towards each mentioned faction. See item_guidelines for the formatting requirements.

 

require_guild_grade: Minimum guild rank needed to access current mission (Leader, HighOfficer, Officer, Member).

 

require_guild_membership (Boolean): Whether the player has to be in a guild or not.

 

require_mission_done: List of mission that have to be completed by the player before accessing current mission. See item_guidelines for the formatting requirements.

 

require_mission_not_done: If the player has completed one mission of this list, he won’t be able to access current mission. See item_guidelines for the formatting requirements.

 

require_mission_not_running: None of the listed mission should be running in order to access current mission. See item_guidelines for the formatting requirements.

 

require_mission_running: All listed mission should be running in order to access current mission. See item_guidelines for the formatting requirements.

 

require_own_item: The player has to have listed items in his inventory in order to access current mission. See item_guidelines for the formatting requirements.

 

require_season: Current mission is only available during listed seasons.

 

require_skill: TBA

 

require_team_size: The player must be part of a group with a at least this number of members to access current mission.

 

require_title: The player must display a given title to access current mission.

 

require_wearing_item: The player must wear all listed items to access current mission. He can’t remove them during the mission without canceling it. See item_guidelines for the formatting requirements.

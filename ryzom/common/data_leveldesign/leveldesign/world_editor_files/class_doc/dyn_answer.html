dyn_answer

 

Answer that is proposed to the player during a dyn_chat step. Opens a new script branch.

 

Properties :

 

name : Name of the answer only used by the mission designer.

 

Phrase_answer : Text displayed in the interface to be clicked by the player to validate this answer. See text guidelines for the formatting requirements.

 

 

 

Also see the step_dyn_chat class.

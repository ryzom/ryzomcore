teleport

Action used to activate a teleport option on a NPC that will be accessible via the contextual menu. It will be deleted when the mission ends.

 

Properties:

 

name: Action name, only used by the mission designer.

 

npc_name: Tag of the NPC that talks.

 

once (Boolean): Deletes the teleport option after its first use.

 

phrase_menu (Optional): Text that appears in the NPC contextual menu. See text guidelines for the formatting requirements.

 

world_position: Teleport target coordinates. Its format is “<x_coordinate>,<y_coordinate>”.

 

Default variables:

 

phrase menu (place):     $p$

recv_faction_point

 

Allow the mission to give or remove faction points to the mission’s owner.

 

Properties:

 

name: Action name, only used by the mission designer.

 

faction: Specify which faction points will be increased or decreased. The faction must be one of those: Kami, Karavan, Fyros, Matis, Tryker or Zorai

 

point: The number of points you will give to the mission’s owner (if the number is positive) or remove (if the number is negative).
